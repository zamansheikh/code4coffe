How to send code in discord ;

```languageNameExtension

    code{
        //code
    }
```   
Example:
```c
    #include <stdio.h>
    int main()
    {
        printf("Hello World");
        return 0;
    }
```
To compile:
use ;compile before the code
Example:
;compile
```c
    #include <stdio.h>
    int main()
    {
        printf("Hello World");
        return 0;
    }
```
